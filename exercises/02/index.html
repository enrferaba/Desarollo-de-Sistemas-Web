Tema 2 · Ejercicio 02

Hoja de estilos base para nuestra página de perfil

Seguimos exactamente el guion de clase: reutilizamos el HTML del ejercicio 01 y añadimos la capa de estilos que el profesor explicó en el Tema 2 para practicar selectores, cajas flexibles y jerarquía visual.

1. Qué pide el enunciado

El objetivo es transformar la página de presentación del ejercicio 01 en una maqueta cuidada aplicando los fundamentos de CSS vistos en el Tema 2:

  • Crear una hoja de estilos que defina tipografía, colores base y espaciados consistentes.
  • Organizar el contenido principal con un contenedor centrado y secciones bien diferenciadas.
  • Utilizar Flexbox o Grid sencillo para distribuir bloques como intereses o datos de contacto.
  • Respetar la semántica del HTML original (cabecera, secciones, pie de página) sin añadir scripts.
Recordatorio del profesor: «No hace falta inventar nada nuevo. Copiad el HTML del ejercicio 01, enlazad vuestro CSS y comprobad que el diseño se adapta a móvil y escritorio ajustando márgenes y tamaños de letra.»

2. Paso a paso recomendado en clase

  1. Crear un archivo style.css (o usar un bloque <style> como en este ejemplo).
  2. Definir colores y tipografías globales con variables CSS para mantener coherencia.
  3. Añadir un contenedor central con max-width y márgenes automáticos.
  4. Aplicar display: flex o grid a las secciones que contengan listas o tarjetas.
  5. Usar @media queries sencillas para mejorar la lectura en pantallas grandes.
  6. Revisar contraste, jerarquía de titulares y espaciados antes de entregar.

3. Guía visual y paleta propuesta

Puedes seguir esta referencia de colores y estructura para mantener el estilo uniforme con el resto del equipo.

#1F2937 Cabeceras y textos fuertes
#2563EB Enlaces y acentos
#FBBF24 Notas y llamadas de atención
#F3F4F6 Fondo general

Cabecera

Incluye el título principal y una breve descripción. Usa un fondo oscuro y texto claro.

Sección "Sobre nosotros"

Combina imagen y texto. Puedes usar display: grid con dos columnas a partir de 768px.

Lista de intereses

Transforma la lista en tarjetas con bordes suaves para que cada punto destaque.

4. Ejemplo de código explicado en clase

Este fragmento resume las propiedades clave que deberías tener en tu hoja de estilos. Adáptalo a tus propios colores y tamaños. 

body {
  font-family: "Segoe UI", sans-serif;
  background: #f3f4f6;
  color: #1f2937;
  margin: 0;
}

main section {
  background: white;
  border-radius: 16px;
  padding: 2rem;
  box-shadow: 0 12px 28px rgba(15, 23, 42, 0.08);
}

.intereses {
  display: grid;
  gap: 1rem;
}

@media (min-width: 768px) {
  .intereses {
    grid-template-columns: repeat(3, 1fr);
  }
}

5. Checklist antes de entregar

  • La página se sigue viendo bien al reducir la ventana por debajo de 480px.
  • Los encabezados mantienen la jerarquía h1 → h2 → h3.
  • Los colores tienen contraste suficiente (usa la herramienta recomendada en clase si dudas).
  • No hay estilos inline que rompan la consistencia con el resto del equipo.
  • El archivo está guardado en exercises/02/index.html y enlaza correctamente la hoja de estilos.
